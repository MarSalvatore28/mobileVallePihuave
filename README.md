# TaskMaster

Aplicación móvil híbrida para gestión de tareas diarias.

## Descripción

TaskMaster es una aplicación desarrollada con Ionic y React que permite a los usuarios gestionar sus tareas cotidianas de manera simple y eficiente.

## Funcionalidades

- Agregar nuevas tareas
- Marcar tareas como completadas
- Eliminar tareas
- Persistencia local de datos

## Tecnologías

- Ionic Framework
- React
- Capacitor
- TypeScript
